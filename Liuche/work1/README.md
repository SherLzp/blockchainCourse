# 作业一

#### 一、代码运行说明

1.平台：建议Mac OS或Linux。如果用Windows运行，中文的哈希值和在线验证的结果不一致（英文是一致的）。原因是Windows中中文采用GBK编码，而Mac OS和Linux采用的是UTF-8编码，我想在线验证中文应该也是采用的UTF-8编码，因此中文加密后的哈希值才会不一致。

2.运行方法：直接编译（c++11标准）运行cpp文件，根据命令行提示输入消息即可。
